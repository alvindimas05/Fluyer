## Features Progress

### Basic Functionality
- [x] Play & Pause 
- [x] Next
- [ ] Previous
- [x] Volume Control
- [x] Mute
- [ ] Shuffle & Repeat
- [x] Playlist

### Advanced Functionality
- [ ] Search
- [ ] Group by Folder
- [x] Lyrics
- [x] Animated Background
- [ ] Cover Art Caching

### API Functionality
- [x] MusicBrainz Integration (For Cover Art)
- [ ] Spotify Integration (For Spotify Canvas)
- [ ] Soulseek Integration (We love FLACs <3)
