## Reporting a Vulnerability

You can report vulnerabilities publicly or privately via GitHub:

- **Private:** [Open a security advisory](https://github.com/alvindimas05/Fluyer/security/advisories/new)
- **Public:** [Open an issue](https://github.com/alvindimas05/Fluyer/issues/new)

### Response Time

If I'm actively working on the project, I'll respond as soon as possible.
You can check recent commit activity to get a sense of that. If the repo has
been quiet for a while, I can't guarantee a quick response, but I will
eventually get to it.

Once acknowledged, a fix or update will be provided depending on the complexity
of the issue, but shouldn't be more than **2 weeks** after acknowledgment.
